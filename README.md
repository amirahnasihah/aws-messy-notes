# Standard Repo Template

This is the official template for new projects in the Doktahoo organization. It includes:

- Custom issue templates for structured reporting (Additions, Fixes, etc.).
- Starter GitHub workflows for CI/CD.
- With Claude code actions and GitHub actions (cost-optimized).

## Claude Code Actions

See [.github/docs/CLAUDE_ACTIONS.md](.github/docs/CLAUDE_ACTIONS.md) for workflows, flow overview, and cost optimizations.

## How to Use

1. Click "Use this template" to generate a new repo.
2. Customize as needed.
3. Delete this README if it's a fresh project.
4. Link the repository to the project management dashboard
